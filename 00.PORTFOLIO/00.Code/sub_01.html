EDIYA COFFEE

메인페이지를 직접 설계하고 마크업하여 제작한 이디야 커피 프로젝트 입니다.
기업 사이트와 브랜드 사이트의 내용 구분이 정확히 되어있지 않은 점
메인에서 신상품, 베스트 메뉴 확인이 어려운 점, 정적인 효과를 개선하려고 했습니다.

HTML / CSS / JavaScript

DESIGN

◀︎ ☕ INTRO

브랜드와 기업 사이트를 구분함
해당 페이지에 마우스를 올리면 배경이 밝아지며 영상이 재생됨

▶︎ ☕ ️MAIN

메뉴의 중간에 이디아 로고를 넣어 눈에 띄게 함
메뉴에 마우스를 올리면 이디야에 대한 메뉴가 나오게 하여
브랜드를 더욱 강조시킴

▲ ☕ ️NEW MENU

상단에 새로운 메뉴 타이틀을 움직이게 하여 메뉴 타이틀을 강조
이미지, 방향아이콘, 슬라이드 바, 이미지 설명 내용을 이동 방향으로
자동으로 움직이게 하여 단조로움을 없앰

▲ ☕ ️BEST MENU

해당 메뉴를 클릭하면 메뉴가 뒤집히며 설명이 나오게 구현

◀ ☕ ️ABOUT

이디야에 대한 내용을 넣어 기업의 가치를 강조함
해당 내용의 2/3 높이를 지나면 점점 뚜렷해지며 이미지와 내용이 올라옴

◀ ☕ ️NEWS & EVENT

해당 메뉴에 hover시 배경색, 글씨색 변경됨

◀ ☕ ️FOOTER

이디야 로고를 페이지 전체에 크게 svg로 삽입하여
브랜드 컬러와 로고를 강조
